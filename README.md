# exp395-xrf
Python code with assistance from Claude (Anthropic) for processing and visualizing XRF core scanning data from IODP Expedition 395 (Reykjanes Ridge). Includes elemental downhole plots and cross plots for all sites.

The formatting and correlation figure code were adapted from the R code published by Robustelli Test and Koorapati (2023) and available at https://github.com/Ravikiran2316/IODP-Exp.-390-393-XRF.

Please mention the author (Claire Jasper) when reproducing the code for your analysis. Year: 2026
